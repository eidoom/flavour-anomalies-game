# flavour-anomalies-game

A walking simulator.

Installation:

```shell
git clone https://github.com/eidoom/flavour-anomalies-game.git
```

Usage:

```shell
cd flavour-anomalies-game
python3 main.py
```

Instructions for options:

```shell
python3 main.py -h
```

Dependencies:

* `python` 3.6+

* Python module [`termcolor`](https://pypi.org/project/termcolor/)
